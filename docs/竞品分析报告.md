## 一、市场总览

### 1.1 市场格局

装箱/排柜软件市场呈明显的 **"大场景集中，小场景真空"** 态势：

- **主干赛道（拥挤）**：海运集装箱装柜优化（20GP/40GP/40HC），面向物流/货代行业。国内有悠闲装箱、懒人装箱、CX7 等至少 5+ 款竞品，国外有 3DBinPacking、3DPACK.ING、DeepPack 等 SaaS 产品。
- **细分赛道（空白）**：工厂内部标准木箱的纸箱装箱优化。几乎没有针对性的产品，现有工具无法直接满足"纸箱 → 木箱"的垂直场景。

### 1.2 市场驱动力

| 驱动力 | 说明 |
|--------|------|
| 制造业降本 | 标准木箱利用率每提升 5%，每年可为中型工厂节省 10~30 万元包装成本 |
| 数字化转型 | 从人工试装到算法优化，是包装环节自动化的必然趋势 |
| 数据安全需求 | 工厂不愿将纸箱尺寸、产品清单等敏感数据上传第三方平台 |
| 离线场景 | 车间/仓库网络条件不稳定，本地单文件方案有天然优势 |

## 二、竞品详细分析

### 2.1 悠闲装箱 (optimalpacking.cn)

**定位**：国内领先的免费在线集装箱装柜计算平台  
**目标用户**：外贸、物流、货代行业  
**价格策略**：完全免费，注册即用

**功能矩阵**：

| 功能 | 支持情况 | 备注 |
|------|---------|------|
| 集装箱装箱(20GP/40GP/40HC) | ✅ | 海运箱型全覆盖 |
| 纸箱/托盘/木箱/散货 | ✅ | 多种包装形式 |
| 3D可视化 | ❌ | 据官网信息未明确提供 |
| 混装计算 | ✅ | 多规格货物混装 |
| 木箱反推 | ❌ | 不支持 |
| 批量对比 | ❌ | 不支持 |

## 四、差异化优势分析

### 4.1 本项目独有功能（竞品零覆盖）

| 功能 | 市场需求强度 | 技术壁垒 | 竞品跟进难度 |
|------|-------------|---------|------------|
| **木箱反推** | ★★★★☆ | ★★★☆☆ | 中（需改算法方向） |
| **批量对比** | ★★★★★ | ★★☆☆☆ | 低（UI 工作为主） |
| **规格库管理** | ★★★★☆ | ★☆☆☆☆ | 低 |
| **本地单文件** | ★★★★☆ | ★★☆☆☆ | 低 |

### 4.2 差异化价值定位

```
集装箱场景（竞品争夺）                木箱场景（本项目）
─────────────────────  ─────────────────────
问题：集装箱怎么装最省空间          问题：这批纸箱用什么规格的木箱最合适
用户：货代/物流经理                  用户：工厂包装工程师/仓库管理员
核心指标：集装箱利用率                核心指标：木箱数量 + 利用率综合最优
决策：选什么柜型                       决策：选什么木箱尺寸
竞争烈度：红海（5+竞品）               竞争烈度：蓝海（几乎无竞品）
```

### 4.3 潜在竞争威胁

| 威胁来源 | 威胁等级 | 应对策略 |
|---------|--------|---------|
| 悠闲装箱扩展到木箱场景 | 🟡 中 | 木箱反推是算法难度壁垒，不是 UI 能快速复制的 |
| CX7 增加木箱模块 | 🟡 中 | CX7 的数据加密+重心校验是优势，但定价较高 |
| DeepPack 降低门槛 | 🟢 低 | 目前面向高端市场，暂不构成直接竞争 |
| 开源算法库（如 py3dbp） | 🟢 低 | 需要 Python 环境，无法开箱即用 |

## 五、发展建议

### 5.1 短期（2~4周）

1. **重量约束（V2 P0）**：改动量小，补齐与 CX7 的功能差距
2. **产品化包装**：完善 README 和在线演示，方便目标用户了解功能
3. **性能优化**：纸箱数量多时，避免单线程卡顿

### 5.2 中期（1~2个月）

1. **模块拆分 + 构建工具**：提升可维护性，降低后续功能迭代成本
2. **PWA/Service Worker**：实现离线可用，强化"本地部署"优势
3. **补测试**：算法核心 100+ 单元测试，建立质量基线

### 5.3 长期（3~6个月）

1. **导出格式扩展**：PDF 报告 + 装箱指导图、BOM 清单
2. **AI 辅助优化**：根据历史装箱数据，推荐最优木箱规格
3. **企业版功能**：多用户协作、方案审批流程、ERP 对接 API
4. **定价策略**：免费版（基础功能）+ 企业版（高级功能/API）

---

> **总结**：本项目在"木箱装箱"这个垂直场景中拥有显著的首发优势和功能独特性。竞品要么面向不同场景（集装箱），要么功能残缺（不能混装/无 3D）。如果能在 2~4 周内补齐重量约束，并启动模块化拆分，将建立起可持续的竞争壁垒。
